# Compumur

Welcome to another operating system. This time, we would ask all Symbian community to help us to revitalize this magical system for PC.

**What is Compumur?**

Just a retry of **Symbian**, the magical OS defunct since last decade. But this time, for PC. All we want is to recreate the basic features that characterised it: fast, light and capable to run on low hardware. An OS from everyone, to everyone.

**Is it then Linux?**

No at all! (But any help from there will be also appreciated!) We try to make an independent OS for running on any computer.

**Will it work soon?**

Depends on how large is the support. Anyone can contribute, so we would thank any code line lent here. You should share this with all those you know from Symbian community, as we could release it before!

**If I understood correctly... Is this just a Symbian PC OS?**

We hope to be something more that that on the future! This will start when you too!
